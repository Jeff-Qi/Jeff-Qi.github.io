MyCat

简介

  • mycat是一个数据库中间件，也可以理解为是数据库代理。在架构体系中是位于数据库和应用层之间的一个组件，并且对于应用层是透明的，即数据库感受不到mycat的存在，认为是直接连接的mysql数据库（实际上是连接的mycat,mycat实现了mysql的原生协议）
  • mycat的三大功能：分表、读写分离、主从切换

安装使用

  2. 下载安装 MyCat
