docker machine

简介

  • 短时间内在本地或云环境中搭建一套 Docker 主机集群；负责实现对Docker 运行环境进行安装和管理
  • 基本功能
    1. 在指定节点或平台上安装Docker 引擎，配置其为可使用的Docker 环境；
    2. 集中管理（包括启动、查看等）所安装的 Docker 环境
